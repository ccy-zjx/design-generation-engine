# Design Generation Engine (DGE)

> Instead of generating pixels, we generate **design structure**.

## Overview
Design Generation Engine is a multi-agent AI system that converts design references into structured layouts and PSD files.

## Architecture
Input → Analysis → Planning → Layout → JSON → Visual → PSD

## Features
- Multi-agent workflow  
- Structured design generation  
- PSD export
